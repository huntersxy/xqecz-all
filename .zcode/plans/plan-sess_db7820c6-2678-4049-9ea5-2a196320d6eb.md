# 实现计划

## 需求总结

1. **瀑布流主题作为默认主题**（所有设备）
2. **首页卡片显示缩略图，点击后加载原图**
3. **缩略图不再裁切**，宽度保持 800px
4. **批量重新生成缩略图**支持全部重新生成

---

## 1. 修改默认主题为瀑布流

**文件**: `frontend/src/stores/theme.ts`

将默认主题从 `default` 改为 `waterfall`，移除屏幕宽度判断逻辑：

```typescript
// 修改前
if (savedTheme && savedTheme !== 'default') {
  currentTheme.value = savedTheme
} else {
  currentTheme.value = window.innerWidth >= 1024 ? 'bilibiliStyle' : 'default'
}

// 修改后
currentTheme.value = savedTheme || 'waterfall'
```

---

## 2. 修改瀑布流主题图片加载逻辑

**文件**: `frontend/src/themes/WaterfallTheme.vue`

将首页卡片图片从原图改为缩略图，点击后加载原图：

### 2.1 卡片显示缩略图
```vue
<!-- 修改前 -->
<img :src="getImageUrl(item.img || item.thumb)" />

<!-- 修改后 -->
<img :src="getImageUrl(item.thumb)" />
```

### 2.2 点击卡片加载原图
在卡片点击事件中，跳转到详情页时传递原图 URL，或者在详情页中使用 `img` 字段。

---

## 3. 修改缩略图生成逻辑（不再裁切）

**文件**: `internal/util/video.go`

### 3.1 修改 `GenerateImageThumbnail`
```go
// 修改前：裁切为 4:3 + 缩放
-vf "crop=...,scale=800:-1"

// 修改后：只缩放，不裁切
-vf "scale=800:-1"
```

### 3.2 修改 `GenerateVideoThumbnail`
同样移除 crop 滤镜，只保留缩放：
```go
// 修改前
-vf "select=eq(n\,9),crop=...,scale=800:-1"

// 修改后
-vf "select=eq(n\,9),scale=800:-1"
```

---

## 4. 批量重新生成缩略图（支持全部重新生成）

**文件**: `internal/service/admin.go`

### 4.1 修改 `RegenerateAllThumbnails` 方法
当前只处理 `thumb_path IS NULL OR thumb_path = ''` 的内容，需要改为处理所有有文件的内容：

```sql
-- 修改前
SELECT ... WHERE deleted_at IS NULL AND file_path != '' 
  AND (thumb_path IS NULL OR thumb_path = '') 
  AND (type = 'image' OR type = 'video')

-- 修改后
SELECT ... WHERE deleted_at IS NULL AND file_path != '' 
  AND (type = 'image' OR type = 'video')
```

### 4.2 在重新生成前删除旧缩略图
在生成新缩略图前，先调用 `util.DeleteThumbnail` 删除旧文件。
